Method: Highlight a Segment of Code

Parameters:

  • code must be a String.
    (String of code to be highlighted)
  • lang is optional, must be a String, and has a default value of 'AUTO'.
    (Language of the code, will default to auto detection)

Returns a String
(Highlighted code (HTML))
